## Prerequisites for First Production Deploy
1. Add GitHub secrets: `SSH_HOST`, `SSH_USER`, `SSH_KEY` (and optionally `SSH_PORT`)
2. Set up cron job for backups: `sudo crontab -e` → `0 3 * * * /usr/local/bin/interxchange-backup`
3. Run `bash deploy/deploy.sh` from project root for initial production deploy
4. Ensure `.env` has production-strength secrets (use `deploy/.env.production` as template)
